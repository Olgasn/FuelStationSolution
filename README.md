# FuelStationSolution
Учебный пример простого REST Web-приложения баз данных на языке C#, использующего такие технологии и инструменты как: 
чистая архитектура, CQRS, ASP .NET Core 10, EF Core 10, Swagger, SQLite, Fluent Validation, XUnit, Serilog, Automapper, MediatR.
